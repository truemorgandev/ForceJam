FORCEJAM

Tactical Multitool for Ukrainian Armed Forces

About the Project

FORCEJAM is a portable device designed to detect and jam FPV drone signals, monitor radio frequencies, and securely transmit encrypted messages between soldiers, bypassing internet and cellular networks.

Main Features

  • Real-time monitoring of 433MHz, 868MHz, and 2.4GHz frequencies
  • Manual drone jamming using nRF24L01+ PA/LNA and CC1101 modules
  • Encrypted messaging with AES-256 protocol and screen keyboard input

Hardware

  • ESP32 microcontroller
  • Color 4-inch touchscreen TFT display
  • 3× nRF24L01+ PA/LNA modules
  • 2× CC1101 modules for 433/868 MHz
  • Rechargeable Li-Po battery

Project Values

  • Support for Ukrainian forces
  • Open source and community-driven
  • Affordable and modular design
